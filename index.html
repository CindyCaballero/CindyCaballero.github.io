Actividad No. 2 Creando la Plantilla con HTML5 y CSS3

El estudiante creará un sitio web utilizando lenguajes de diseño como HTML5, CSS3 y el cual servirá como plantilla para el diseño y publicación de un Objeto Virtual de Aprendizaje OVA, con base a los parámetros y elementos establecidos en la guía integrada de actividades del curso.

HTML y el CSS son lenguajes simples. Las últimas versiones de estos lenguajes, HTML5 y CSS3, nos ofrecen actualmente posibilidades increíbles. Con ellos podréis añadir texto a un sitio web, construir un menú de navegación, añadir imágenes, efectos gráficos y mucho más.

By Cindy P. Caballero Arrieta
